# Mi App Flutter - Entregable Individual

Calculadora de **Índice de Masa Corporal (IMC)** desarrollada en Flutter como entregable individual del taller.

## ¿Qué hace?

Recibe peso (kg) y estatura (cm), calcula el IMC y muestra la categoría: bajo peso, normal, sobrepeso u obesidad.

## ¿Para qué se utiliza?

Auto-evaluación rápida del estado nutricional. Útil en consulta de primer contacto o seguimiento personal.

## Cómo correrla

```bash
flutter pub get
flutter run
```
